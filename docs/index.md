# Razor Designer

> One grain at a time, we fill the s&box.

**Razor Designer** is an in-editor visual designer for `.razor` UI panels in s&box.
Drag from a component palette, arrange in a live canvas, edit properties in an
inspector, and save out clean `.razor` + `.razor.scss` pairs that drop straight
into your project.

[Get Started :material-arrow-right:](getting-started/installation.md){ .md-button .md-button--primary }
[View on GitHub :fontawesome-brands-github:](https://github.com/obselate/Grains){ .md-button }

---

## What you get

- **Palette** — drag built-in components and saved templates onto the canvas.
- **Hierarchy** — reparent, reorder, and navigate the control tree.
- **Canvas** — live preview with theme presets (github-dark, gruvbox-dark, solarized-dark).
- **Inspector** — edit properties, lengths, classes, and bindings.
- **Templates** — save reusable groups of controls.
- **Round-trip serialization** — outputs `.razor` and `.razor.scss` you can edit by hand.
