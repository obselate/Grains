## Add to your s&box project

1. Open your `.sbproj` in the s&box editor.
2. Add the `xaz.razordesigner` library as a referenced project (or place it under
   your project's `Libraries/` folder).
3. Reload the editor — the **Razor Designer** window appears under the editor's
   tools menu.

## Open the designer

Open the **Razor Designer** window from the editor. You should see four panels:

- Palette (left)
- Hierarchy (left, below palette)
- Canvas (center)
- Inspector (right)

Continue to [Your First Panel](first-panel.md).
