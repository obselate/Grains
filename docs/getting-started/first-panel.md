# Your First Panel

!!! note "Stub"
    Walk through building a small panel end-to-end. Add screenshots/GIFs.

## 1. Create a new document

Open the Razor Designer window and start a fresh document.

## 2. Drag from the palette

Pick a starting template (e.g. `ExampleCard`) from the **Palette** and drop it
on the **Canvas**.

## 3. Edit in the inspector

Select the control in the **Hierarchy**, then tune properties in the
**Inspector** — text, classes, lengths, etc.

## 4. Save

Save the document — Razor Designer writes a paired `.razor` and `.razor.scss`
into your project.

## 5. Use in-game

Reference the panel from your `RootPanel` or any parent panel, exactly like any
hand-written `.razor`.
